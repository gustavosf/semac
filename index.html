Tema / Funcionalidades Básicas

Sistema de inscrição e frequência de alunos para a Semana Acadêmica do Instituto.

Funcionalidades mínima:

  • um site onde os alunos se inscrevam na atividade da SEMAC desejada
  • o sistema:
    • verifica colisões de horários entre atividades para cada aluno
    • fornece lista de inscritos para o professor responsável pela atividade
    • permite a comex e as comgrads acompanhar todo o processo de inscrição

Tema / Funcionalidades Complementares

Opcionalmente, um conjunto complementar de funcionalidades inclui:

  • controle de frequência on-line (para analisar e discutir a melhor forma - para atividades em laboratório pode ser o login, para seminarios e palestras, não tenho ideia)
  • relatórios de frequência por atividade e por aluno
Detalhes adicionais podem/devem ser obtidos diretamente com o cliente.

Ferramentas

FuelPHP http://fuelphp.com

Fuel is a simple, flexible, community driven PHP 5 web framework. It was born out of the frustrations people have with the current available frameworks and developed with support from a community of developers. Fuel is extremely portable, works on almost any server and prides itself on clean syntax.

Bootstrap, from Twitter http://twitter.github.com/bootstrap

Bootstrap is a toolkit from Twitter designed to kickstart development of webapps and sites. It includes base CSS and HTML for typography, forms, buttons, tables, grids, navigation, and more.
